# PhD Habit Tracker

A long-term, truth-based habit tracker designed for PhD life.

## Features
- Daily night-time review
- Dynamic habits (add, pause, evolve)
- Minimum-threshold completion logic
- Multi-year tracking
- Weekly / long-term pattern analysis
- Local-first storage with export

## How to deploy on GitHub Pages

1. Create a new GitHub repository
2. Add a file named `index.html`
